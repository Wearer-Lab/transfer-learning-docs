## Installation

Get up and running with Transfer Learning quickly by following these steps.

### Prerequisites

Before installing Transfer Learning, make sure you have:

- Python 3.9 or higher
- An OpenAI API key (for GPT-4 Vision)
- FFmpeg installed on your system (for video processing)

<AccordionGroup>
  <Accordion icon="key" title="Setting up your OpenAI API key">
    1. Create an account on [OpenAI](https://platform.openai.com/) if you don't have one
    2. Navigate to the API section and create a new API key
    3. Copy your API key for use during installation
  </Accordion>
  <Accordion icon="film" title="Installing FFmpeg">
    **On macOS (using Homebrew):**
    ```bash
    brew install ffmpeg
    ```
    
    **On Ubuntu/Debian:**
    ```bash
    sudo apt update
    sudo apt install ffmpeg
    ```
    
    **On Windows:**
    Download from [FFmpeg's official website](https://ffmpeg.org/download.html) or install using Chocolatey:
    ```bash
    choco install ffmpeg
    ```
  </Accordion>
</AccordionGroup>

### Installation Steps

<Steps>
  <Step title="Install Transfer Learning">
    Install using UV (recommended):
    
    ```bash
    pip install uv
    uv pip install transfer-learning
    ```
    
    Or using pip:
    
    ```bash
    pip install transfer-learning
    ```
  </Step>
  <Step title="Set up your API key">
    Create a `.env` file in your working directory:
    
    ```bash
    echo "OPENAI_API_KEY=your-api-key-here" > .env
    ```
    
    Or set it as an environment variable:
    
    ```bash
    export OPENAI_API_KEY=your-api-key-here
    ```
  </Step>
  <Step title="Verify installation">
    Run the following command to verify that Transfer Learning is installed correctly:
    
    ```bash
    transfer-learning --version
    ```
    
    You should see the version number displayed.
  </Step>
</Steps>

## Quick Examples

Here are some quick examples to get you started with Transfer Learning.

### Process a Local Video

Extract frames from a local video file and analyze their content:

```bash
transfer-learning process-video path/to/video.mp4
```

### Generate a Guide from Processed Video

Generate a step-by-step guide from previously processed video data:

```bash
transfer-learning generate-guide data/video_name
```

### Process a YouTube Video

Download and process a YouTube video:

```bash
transfer-learning process-youtube "https://www.youtube.com/watch?v=VIDEO_ID"
```

### Complete YouTube Pipeline

Download a YouTube video and generate a guide in one command:

```bash
transfer-learning youtube-guide "https://www.youtube.com/watch?v=VIDEO_ID"
```

## Troubleshooting

<AccordionGroup>
  <Accordion icon="triangle-exclamation" title="OpenAI API key issues">
    If you encounter errors related to the OpenAI API key:
    
    1. Verify that your API key is correct
    2. Check that your API key has sufficient credits
    3. Ensure the API key is properly set in your environment or `.env` file
  </Accordion>
  <Accordion icon="video" title="Video processing issues">
    If you encounter issues with video processing:
    
    1. Verify that FFmpeg is installed and accessible in your PATH
    2. Check that the video file is not corrupted
    3. Try processing with a smaller frame interval: `--frame-interval 60`
  </Accordion>
  <Accordion icon="memory" title="Memory or performance issues">
    If you encounter memory or performance issues:
    
    1. Reduce batch size: `--batch-size 5`
    2. Reduce concurrent batches: `--max-concurrent 2`
    3. Increase frame interval to extract fewer frames: `--frame-interval 60`
  </Accordion>
</AccordionGroup>
